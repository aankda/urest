# Changelog
All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog.

## [1.0.1] - 2026-03-16
### Added
- Local stats counter (“Blocks Neutralized”) stored on-device.
- Context menu toggle to disable/enable URest per site.
- Handshake status indicator to confirm active injection.
- JSON‑LD + Open Graph metadata for the landing page.
- Third‑party notices file for font/icon attributions.

### Changed
- Popup icon updated to the official URest icon.
- Landing page hero simplified for a cleaner, macOS‑like feel.
- Store status messaging simplified for a cleaner public presentation.
- Fonts are self‑hosted for privacy‑pure, offline‑friendly delivery.

### Fixed
- Privacy policy updated to reflect current permissions and local‑only behavior.
